# Run Length Encoding

A simple implementation of the Run Length Encoding (RLE) algorithm in Java. This project encodes strings by compressing consecutive occurrences of characters.

## Table of Contents

- [Features](#features)
- [Installation](#installation)

## Features

- Efficient compression of strings with repeated characters.
- Handles various input cases, including empty strings and single characters.
